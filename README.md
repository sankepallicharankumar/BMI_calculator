# BMI Calculator 🏋️‍♂️

A simple **Body Mass Index (BMI) Calculator** that calculates BMI based on a user's height and weight and displays the health category.

## 📌 Features
- Enter height and weight
- Calculate BMI instantly
- Displays BMI category:
  - Underweight
  - Normal weight
  - Overweight
  - Obesity
- Simple and user-friendly interface

## 📊 BMI Formula

BMI is calculated using:

\[
BMI = \frac{Weight\ (kg)}{Height^2\ (m^2)}
\]

Example:

Weight = 70 kg  
Height = 1.75 m  

BMI = 70 / (1.75 × 1.75) = **22.86**

## 🏷️ BMI Categories

| BMI Range | Category |
|----------|----------|
| Below 18.5 | Underweight |
| 18.5 – 24.9 | Normal weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and above | Obesity |
